
# Collexa – PWA Campus Event Platform

Collexa is a production-ready Progressive Web App (PWA) designed to bring everything a campus needs into one unified platform.

---

## What is Collexa?

Collexa is an innovative platform that connects students and event organizers within a structured digital ecosystem.

In most colleges, event information is scattered across multiple channels — WhatsApp groups, posters, social media, and word of mouth. This fragmentation often causes students to miss valuable opportunities simply because they were unaware.

Collexa solves this by acting as a **centralized hub for campus events and collaboration**.

---

## Why Collexa?

Collexa is built around real-world student problems:

-  Missed opportunities due to poor visibility  
-  Unorganized event announcements  
-  Difficulty in forming and managing teams  
-  Lack of a centralized campus platform  

 Collexa addresses all of these through a **fully integrated digital experience**

---

## How Collexa Works

### For Students
- Discover all campus events in one place  
- View complete event details  
- Register instantly  
- Join or create teams  
- Track participation  

### For Admins
- Create and manage events  
- Monitor registrations  
- Control event flow  

---

## What Makes Collexa Different?

Collexa goes beyond a typical event listing platform.

It creates a **complete campus ecosystem** by:

- Connecting students and organizers  
- Encouraging collaboration  
- Eliminating information gaps  
- Strengthening campus engagement

## Features

-  Firebase Authentication (Login/Register)
-  Role-based Protected Routes (Student/Admin)
-  Event browsing & registration
-  Team participation system
-  Admin dashboard
-  Calendar view
-  Offline support (PWA)
-  Installable application
-  Fast, responsive UI

---

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite  
- **Styling:** Tailwind CSS + Framer Motion  
- **State Management:** React Context API  
- **Backend/Database:** Firebase (Auth + Firestore)  
- **Hosting:** Firebase Hosting  
- **PWA:** Service Worker + Web App Manifest
