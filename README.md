# Collexa – Campus Event Platform (PWA)

Collexa is a production-ready Progressive Web App (PWA) that helps students discover, join, and manage campus events — all in one place.

---

## Live Demo
https://campus-connect-f1f29.web.app

---

## Features

- Firebase Authentication (Login/Register)
- Protected Routes (Role-based access: Student/Admin)
- Event browsing & registration
- Team participation system
- Admin dashboard for event management
- Calendar view for events
- Offline support (PWA)
- Installable app (Add to Home Screen)
- Fast and responsive UI

---

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite
- **Styling:** Tailwind CSS + Framer Motion
- **State Management:** React Context API
- **Backend/DB:** Firebase (Auth + Firestore)
- **Hosting:** Firebase Hosting
- **PWA:** Service Worker + Web App Manifest
